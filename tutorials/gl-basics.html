What is WebGL?

It is an implementation of OpenGL ES in Javascript

It is an API for fast graphics redering on the GPU

What WebGL is not?

It is not a 3D engine

In fact, it is not related to 3D at all

1. Buffers

2. Shaders

3. Textures
