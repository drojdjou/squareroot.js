Class: Quaternion

SQR.Quaternion()

new Quaternion()

Represents a quaternion with optionally setting the values directly.

Just as a reminder, given an angle a and an axis x,y,z this is what the quaternion values are:

Source:

Example
var q = new SQR.Quaternion();
var s = Math.sin(a / 2);
q.x = x * s;
q.y = y * s;
q.z = z * s;
q.w = Math.cos(a / 2);
